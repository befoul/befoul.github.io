How secure is your machine? lets find out!

When faced with the topic of operating system security, the question arises: do you trust it as it is, or have you gone the extra mile to implement security architectures that make the machine more robust?

HINT: the first option is the worst!

What can you do to secure your machine.?

there are plenty of open source tools you can use to test your machines security rating such as:

Lynis for linux

OSX Auditor for MacOS

OpenSCAP for linux

these tools are loads of help when you're a novice or if you just cant quite remember what all areas you need to secure next. As with every opensource tooling on the internet be sure to look at the documentation before installing and running.
